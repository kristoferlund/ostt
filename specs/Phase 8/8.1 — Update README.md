# 8.1 — Update README

## Problem

The README documents the existing command set but does not cover the new processing feature. Users discovering ostt need to see processing in the feature list, command reference, configuration examples, and quick start.

## Dependencies

- **Phase 5** — CLI integration complete (process subcommand and -p flag exist)

## Objective

Update `README.md` to document the processing pipeline feature end-to-end.

## Specification

### Changes to README.md

#### Features section

Add a bullet to the features list:

```markdown
- **Post-processing pipeline** - Chain AI or bash actions after transcription to clean up text, translate, generate commands, and more
```

#### Commands section

Add the new commands and flags to the command reference block:

```bash
ostt process         # Post-process most recent transcription (show action picker)
ostt process 3       # Post-process transcription #3 from history
ostt process -a clean   # Run the "clean" action on most recent transcription
ostt process --list  # List configured processing actions
ostt -p              # Record, transcribe, then process (show picker)
ostt -p clean -c     # Record, transcribe, run "clean" action, copy to clipboard
ostt record -p clean # Same as above (explicit record subcommand)
ostt transcribe f -p clean  # Transcribe file and process
ostt retry -p clean  # Retry transcription and process
```

Add `p` (process) to the command aliases line.

#### Configuration section

Add a processing configuration example after the existing `[providers.*]` examples:

```toml
# Processing actions
[[process.actions]]
id = "clean"
name = "Clean up text"
type = "ai"
tool = "claude-code"
model = "sonnet"
inputs = [
    { role = "system", content = "Clean up the following transcribed text. Fix grammar, remove filler words, and improve clarity. Return only the cleaned text." },
    { role = "user", source = "transcription" },
]

[[process.actions]]
id = "cmd"
name = "Generate CLI command"
type = "bash"
command = "tr '[:lower:]' '[:upper:]'"
```

#### Quick Start section

Add a brief processing example after the existing recording example:

```markdown
# Process your transcription with an AI action
ostt -p clean -c

# Or process a previous transcription
ostt process -a clean
```
