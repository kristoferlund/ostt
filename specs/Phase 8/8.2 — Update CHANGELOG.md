#### Added

- **Processing pipeline** — Post-process transcriptions using AI CLI tools or bash commands. Configure actions in `ostt.toml` under `[[process.actions]]`.
- **`ostt process` command** — Post-process an existing transcription from history. Supports action picker for interactive selection or `--action` flag for direct execution. Alias: `p`.
- **`-p` / `--process` flag** — Chain processing directly after recording, transcribing, or retrying. Use `-p` to show the action picker, or `-p <action>` to run a specific action.
- **Action picker TUI** — Interactive fullscreen action selection UI, consistent with existing ostt TUI style (history, keywords).
- **AI CLI tool execution** — Execute processing actions via external AI CLI tools (OpenCode, Claude Code, Gemini CLI, Codex CLI). Configurable tool binary and arguments.
- **Bash action execution** — Pipe transcriptions through shell commands for custom processing.
- **Processing animation** — Logo animation with "Processing..." status label during action execution, matching the existing "Transcribing..." animation.

The exact wording should be refined at implementation time to match the actual scope of
what was delivered. If any feature was descoped or changed during implementation, the
changelog should reflect reality, not the original spec.
